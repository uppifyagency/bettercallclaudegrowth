# Cheatsheet — Instagram Performance Marketing

Robert Thomas — Meta/Facebook/Instagram performance ads for conversions. Core thesis: **lower CPM → lower CPA.**

## The Seven Pillars (in order)
1. **Campaign Objective Selection** — biggest lever; Meta picks audiences by the action you optimize for. Wrong objective = higher CPA.
2. **Audience Targeting** — upload 1st-party customers as Custom Audience → build Lookalike; add Google Search Audience Feeds (purchase-intent).
3. **GEO Targeting** — expand countries to drop CPM while keeping converting customers.
4. **Ad Creative** — optimize for CTR; video (incl. AI/UGC-style) wins; enables engagement-based retargeting.
5. **Ad Settings** — option choices (e.g. drop Audience Network) drive more conversions at lower cost.
6. **Landing Page Optimization** — mirroring to cut bounce, lift page conversions.
7. **Post-Ad Engagement** — automations to engage leads the instant they convert/fill a form.

## Campaign Objective → Use → Relative CPM
| Objective | When to use | CPM |
|---|---|---|
| Awareness / Reach | Max impressions, broad audience, no specific action | **Lowest** |
| Engagement / Traffic / Lead Gen | Target users likely to interact / take an action | **Higher** |
| Conversions (Sales, Sign-ups) | Drive purchase, registration, download | **Highest** |

Rule: pick the objective that matches the **exact action** you want (Sales for purchases, Engagement→Lead Forms for leads, App Installs for downloads). Traffic ≠ sales. Conversion objective + **broad audience + wide GEO = net-lower CPM**; conversion + US-only = CPM spikes.

## GEO Targeting
- US/Canada CPM can hit ~$300; expanding to many converting countries can drop CPM **under $1**.
- **Emerging markets** (SE Asia, Africa, Latin America — India, Indonesia, Nigeria): less competition, high engagement, lowest CPMs.
- Localize language/currency/imagery; target **English (All)** when broadening footprint.
- **Go broad:** new product/audience tests, emerging markets, awareness. **Go specific:** retargeting warm audiences, local events, niche products.
- Author's default high-converting country set: **US, Canada, UK, Australia, New Zealand, Germany, France, Italy, Spain, Brazil.**

## Accelerated Ad Spend (core rules)
- **Front-load budget**: algorithm needs data to learn — feed it fast. Set **$500/day** to launch (each campaign if testing several).
- **Learning phase exit = 50 conversion events** (≤7 days; author aims for first 24–48h).
- **Don't accelerate** unless early signs are good: starting CPA within **50–60%** of target (optimized CPA ends ~**40–60% lower** than start).
- **Minimize edits during learning** — major changes reset the phase. Tweak only gradually after exit.
- **Kill rule:** if not performing after ~10k impressions, turn it off — test variations of objective, conversion location, creative, custom audiences, interests, GEO.

## Ad Settings Launch Checklist
- **Campaign objective** → match conversion goal (CPM follows conversion location).
- Setup popup → **Manually Build your Campaign**.
- **Special Ad Categories** → declare if housing/credit/etc.
- **Advantage Campaign Budget** → **OFF** (one ad set; avoid self-competition/higher CPM).
- **A/B Test** → **OFF** at start (skews early data).
- **Conversion Location** → usually Website; track with Pixel.
- **Facebook Page** → select.
- **Performance Goal** → **Maximize Number of Conversions**.
- **Dataset / Conversion Event** → pick Pixel dataset + event.
- **Cost per Result Goal** → leave **blank**.
- **Dynamic Creative** → **OFF**.
- **Budget & Schedule** → **$500/day**; schedule to peak hours (24/7 raises night CPA).
- **Audience Controls** → wide GEO; set min age higher for decision-makers; **English (All)**.
- **Audience+** → use, click **Audience Suggestions (optional)**; add interests + job titles + demographics.
- **Placements** → Manual; **uncheck Audience Network** (lowers conversions 100% of the time).
- **Ads** → connect FB/IG pages, upload creative, CTA matching the action, set destination link, tracking events. Leave Browser Add-Ons/dynamic Languages off.

## The CPM → CPA Conversion Loop
1. **Lower CPM** via wide GEO (high-converting countries) + broad audience.
2. **Feed 1st-party customer data** → Custom Audience → **Lookalike 0–3%**.
3. **Supplement** with Google Search Audience Feeds (400k+ consumers) → Lookalike.
4. **Refine** with manual interests + job titles (original/Advantage+ audience — never auto).
5. **Best creative = highest CTR** (AI/UGC video; CTR up to ~9.8% vs ~0.90% avg) → more value per 1,000 impressions.
6. **Accelerate spend** to clear 50-event learning phase; **track conversions via Pixel** → Lookalikes cut CPA another 10–20%.
7. **Scale to 1,000 conversion events** → lowest CPM, highest CTR, lowest CPA — optimization peak.
